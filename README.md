## рџ“– Overview

IntelLearn is an AI powered web application that helps students discover suitable career paths based on their skills, interests, academic performance, and responses to assessment questions.

Unlike traditional career guidance systems, IntelLearn leverages **TensorFlow.js** to perform machine learning directly in the browser, enabling fast predictions without requiring a Python backend. Firebase is used for secure authentication and cloud database management, while Gemini AI enhances the platform by providing personalized career roadmaps and an intelligent chatbot.

## рџ§  How It Works

1. User registers or logs into the platform.
2. The student completes the career assessment questionnaire.
3. Responses are processed using a TensorFlow.js machine learning model.
4. The system predicts the most suitable career path.
5. Gemini AI generates a personalized learning roadmap.
6. The chatbot assists users with career related questions.
7. Reports are stored securely in Firebase Firestore.
